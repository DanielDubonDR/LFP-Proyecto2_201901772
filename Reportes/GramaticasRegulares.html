Gramáticas Regulares

Gramáticas no cargadas por no ser exclusivamente gramáticas libres de contexto


Gramatica2

No Terminales = { A,B,C,D }
Terminales = {0,1 }
No Terminal Inicial = A
Producciones:


         A -> 1 B
              | 0 C
         B -> 1 A
              | 0 D
         C -> 1 D
              | 0
              | 0 A
         D -> 1 C
              | 0 B



Lineal Izquierda

No Terminales = { A,B,C,D }
Terminales = {0,1 }
No Terminal Inicial = A
Producciones:


         A -> 1 B
              | C 0
         B -> A 1
              | D 0
         C -> D 1
              | 0
              | A 0
         D -> C 1
              | B 0
